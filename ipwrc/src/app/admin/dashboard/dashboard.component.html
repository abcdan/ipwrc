Admin Dashboard

Howdy Admin! Welcome to the admin dashboard. This place gives you more information what you can do as an admin.

Make a product

You can simply add a product by going to the products page and click the "Add new product" button.

Edit a product

You can simply edit a product by hitting the "edit" button under an item from the products page.

View all orders

You can view all orders on this page.

Manage an order

You can manage an order from the order detail page, you can get there by clicking an order from the orders list here.
